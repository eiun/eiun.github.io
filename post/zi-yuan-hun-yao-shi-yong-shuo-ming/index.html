资源混淆使用说明

####工具：

1.使用AabResGuard.jar包进行资源混淆

config.xml中为混淆的配置，需要注意白名单配置避免混淆三方包中的资源，否则可能会导致三方包找不到资源出问题。

merge-duplicated-res为MD5重复的资源合并选项

混淆完成后，当前目录下会生成base-duplicated.txt文件为重复资源目录表，resources-mapping.txt文件为混淆前后资源文件对照表，再次混淆需要删除这两个文件。

混淆命令：

java -jar AabResGuard.jar obfuscate-bundle --bundle=C:\Users\admin\Desktop\test\aab资源混淆\11.aab --output=C:\Users\admin\Desktop\test\aab资源混淆\22.aab --config=config.xml --merge-duplicated-res=true

命令开始后需按确认键继续执行。

2.aab解析成apk命令：

使用bundletool.jar包将aab包的格式解析成apks的格式，供后续手机安装使用。

java -jar bundletool.jar build-apks --bundle=C:\Users\admin\Desktop\test\aab资源混淆\22.aab --output=C:\Users\admin\Desktop\test\aab资源混淆\bb.apks --ks=C:\Users\admin\Desktop\test.jks --ks-key-alias=key0

aab解析成apk后安装命令：

java -jar bundletool.jar install-apks --apks=C:\Users\admin\Desktop\test\aab资源混淆\bb.apks
